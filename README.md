# simple-formater

Modernize code presentation with tab for CSS/HTML/JS formatting

# usage

```sh
用法：./simple-formater -dir <directory>

选项：
  -dir string
        要遍历的根目录路径（必填）
  -h    显示帮助信息（等同于 -help）
  -help
        显示帮助信息

<bug report>    https://github.com/wiselike/simple-formater/issues
```
